
# Setting up live reload for Angular CLI Libraries
A super handy way to make the app reload automatically whether changes are made to the host app or to the library source code, all while keeping the original structure ready to build and publish. <br>
Tutorial: [https://codeeve.com/2020/02/05/setting-up-live-reload-for-angular-cli-libraries/](https://codeeve.com/2020/02/05/setting-up-live-reload-for-angular-cli-libraries/)
